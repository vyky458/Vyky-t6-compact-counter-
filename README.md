# Vyky's Compact T6 Zombie Counter

A clean, compact, and completely transparent zombie counter for Plutonium T6 (Black Ops II Zombies).

## Features
* **Left:** Total remaining zombies in the round.
* **Act:** Currently spawned active zombies on the map.
* **Que:** Remaining zombies waiting to spawn in the queue.
* **Minimalist Design:** Only 1px white border lines with fully transparent backing.
* **Afterlife Fade:** Automatically dims opacity during afterlife mode on Mob of the Dead.

## Installation Guide
1. Download the `zombies_counter.gsc` file from this repository.
2. Navigate to your Plutonium T6 directory: 
   `%localappdata%\Plutonium\storage\t6\scripts\zm`
   *(If the `scripts` or `zm` folders don't exist, simply create them!)*
3. Paste the `zombies_counter.gsc` file inside the `zm` folder.
4. Boot up Black Ops II Zombies and enjoy!
